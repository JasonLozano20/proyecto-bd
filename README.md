## Proyecto de Base de Datos

Este proyecto es una aplicacion para gestionar la base de datos de una bodega de vinos.

## Tecnologías

- NodeJS
- Express
- TypeScript
- Prisma
- MariaDB
- Next.js
- React
- Tailwind CSS
- Prisma Studio
- Electron Forge
- React Native
